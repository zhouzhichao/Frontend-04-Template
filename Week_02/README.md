3、Array(10000).fill(0); // 使用指定的元素填充数组
ES6为Array增加了fill()函数，使用制定的元素填充数组，其实就是用默认内容初始化数组。
该函数有三个参数。

arr.fill(value, start, end)
value：填充值。

start：填充起始位置，可以省略。

end：填充结束位置，可以省略，实际结束位置是end-1。
